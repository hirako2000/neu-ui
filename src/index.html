Philosophy and features

Bootstrap based components system. The kit balances consistency and flexibility to help with rapid prototyping and quick iterative changes when adjusting styles in the browser.

Bootstrap

A responsive and mobile-first theme built with the most popular hence familiar with CSS framework

Build with Sass

Change one variable and the theme adapts. Colors, fonts, sizes, buttons etc.

Responsive Design

Over half of the world web traffic comes from mobile. Neu UI is fully responsive and mobile first

Modular approach

Components overrides and mixins, that can be reused anywhere

Gulp & BrowserSync

Uses Gulp & BrowserSync to offer a seemless and fast workflow for your project

Well documented

The kit is delivered with detailed documentation, hopefully describing all components, styles and usages

PageSpeed

Neu UI is fast, with emphasis on conciseness. CSS effects were limited to keep browsers and bandwidth resource usage low

SVG illustrations

Photography and Illustration SVGs are included, creative common including for commercial use.

Neu design

Each component was crafted with great care to adhere to the Neumorphism look & feel. Using the latest supported CSS features wherever possible
