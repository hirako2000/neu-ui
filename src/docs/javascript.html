Javascript

Here we'll explain where to write your own custom Javascript.

Location

Neu's main javascript file is located at js/neu.js. If you ran one of the npm commands, you'll find another file: neu.min.js which is a minified version of the previous file.

Note: by default, the neu.min.js is the file included and used in the templates.

Content

jQuery or vanilla JS may be use to select elements and work with 3rd party plugins. The entirety of the custom JS is wrapped in a classic jQuery(document).ready(function(){}) function to make sure the page is loaded before the scripts will be running.

For DOM operations you may customize this file or furtherly add other Javascript files to your liking.
